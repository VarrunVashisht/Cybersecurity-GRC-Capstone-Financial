# Sample Cybersecurity Audit Testing

## Control Tested
Multi-Factor Authentication (MFA)

## Control Objective
Prevent unauthorized access to critical systems.

## Control Description
MFA is enforced for all users accessing systems containing customer or financial data.

## Testing Procedure
- Obtained list of systems requiring MFA
- Selected a sample of user accounts
- Verified MFA enforcement
- Reviewed authentication logs

## Test Results
MFA was enabled and enforced for all sampled users.

## Conclusion
Control is operating effectively.

## Risk Rating
Low
